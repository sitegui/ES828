\section{Controlador proporcional}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\begin{par}
Para se obter o tempo mķnimo de estabilizaēćo, os pólos de malha fechadas tem de estar ą mais esquerda possķvel. Observando o root locus da planta:
\end{par} \vspace{1em}
\begin{verbatim}
rlocus(G)
\end{verbatim}
\begin{center}
\begin{figure}[H]
\centering
\includegraphics [width=4in]{pre_relatorio_01.eps}
\end{figure}
\end{center}

\begin{par}
Pode-se observar que isso irį ocorrer quando os dois pólos (verde e azul na figura) se encontrarem, no ponto próximo de -20. Para obter a coordenada exata desse ponto, pode-se verificar a multiplicidade das raķzes da equaēćo caraterķstica $1 + k G(s) = 0$.
\end{par} \vspace{1em}
\begin{par}
No nosso caso, com $G(s) = \frac{P(s)}{Q(s)} = \frac{A}{B s^3 + C s^2 + D s + E}$ , $k(s) = -\frac{B s^3 + C s^2 + D s + E}{A}$.
\end{par} \vspace{1em}
\begin{par}
Como o ponto buscando é raiz dupla de $k(s)$, ele é raiz da derivada de $k(s)$ em $s$, ou seja, raiz de $-\frac{3 B s^2 + 2 C s + D}{A}$.
\end{par} \vspace{1em}
\begin{par}
Substituindo os valores dos coeficientes e resolvendo, chegamos em:
\end{par} \vspace{1em}
\begin{verbatim}
A = G.num{1}(4);
den = G.den{1};
raizes = roots(-polyder(den)/A)
\end{verbatim}

         \begin{verbatim}
raizes =

  -62.2543
  -17.7369

\end{verbatim} \color{black}
    \begin{par}
Observando o grįfico do root locus mostrado anteriormente, sabemos entćo que o valor buscado é $s = -17.7369$. Aplicando
a equaēćo do controlador:
\end{par} \vspace{1em}
\begin{verbatim}
k = -polyval(den, -17.7369)/A
\end{verbatim}

         \begin{verbatim}
k =

    0.8925

\end{verbatim} \color{black}

\begin{figure}[H]
\centering

	\includegraphics [width=6in]{rlocus-C1.jpg}

\end{figure}
\vspace{1em}
\begin{figure}[H]
\centering
	\includegraphics [width=4in]{Step-C1.jpg}
\end{figure}

    \begin{par}
A resposta ą rampa do controlador é exibida abaixo.
\end{par} \vspace{1em}
\begin{verbatim}

t = 0:dt:5;
rampa = t;
sistema = feedback(k*G, 1);
lsim(sistema, rampa, t);

\end{verbatim}

\section{Controlador de Ziegler Nichols}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\vspace{1em}
\begin{par} 
O primeiro passo no método de Ziegler Nichols é encontrar $k = k_{osc}$ que deixa a planta no limiar de estabilidade quando em malha fechada.
\end{par} \vspace{1em}
\begin{par}
Para isso pode ser usado o critério de Routh com a equaēćo caracterķstica:
\end{par} \vspace{1em}
\begin{par}
$$1 + k G(s) = 0$$
\end{par} \vspace{1em}
\begin{par}
$$1 + \frac{8.993 k}{0.0003019 s^3 + 0.03622 s^2 + s} = 0$$
\end{par} \vspace{1em}
\begin{par}
$$\frac{0.0003019 s^3 + 0.03622 s^2 + s + 8.993 k}{0.0003019 s^3 + 0.03622 s^2 + s} = 0$$
\end{par} \vspace{1em}
\begin{par}
$$0.0003019 s^3 + 0.03622 s^2 + s + 8.993 k = 0$$
\end{par} \vspace{1em}
\begin{par}
Com o critério de Routh chega-se em: $k = \frac{0.03622}{0.0003019 * 8.993} = 13.3408$
\end{par} \vspace{1em}
\begin{verbatim}
k_osc = 13.3408;
GC_mf = feedback(G*k_osc,1);
poles = esort(pole(GC_mf));
w_osc = abs(poles(1));
T_osc = 2*pi/w_osc;
kp = 0.6*k_osc;
Ti = T_osc/2;
Td = T_osc/8;
C = kp*(1 + 1/(Ti*s) + Td*s)
\end{verbatim}

         \begin{verbatim}
C =
 
  0.005963 s^2 + 0.4369 s + 8.004
  -------------------------------
             0.05459 s
 
Continuous-time transfer function.

\end{verbatim} \color{black}
    \begin{par}
Usando o sisotool para ajustar o valor de kp e melhorar o desempenho do controlador, obtemos $kp_2 = 1.31 kp$.
\end{par} \vspace{1em}
\begin{par}

\end{par} \vspace{1em}

\end{par} \vspace{1em}
