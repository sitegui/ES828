\section{Objetivos}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\begin{par}
O objetivo deste experimento foi a identificaēćo da funēćo de transferźncia
de um sistema eletrōnico de terceira ordem composto por quatro
estįgios. A identificaēćo dos parāmetros $i > 0, i = 1,...,4$,
$\tau_2 > 0$ e $\tau_3 > 0$ deve ser feita sequencialmente, a cada estįgio,
levando em conta que em é aplicado um sinal de onda quadrada
de frequźncia 1 [Hz] e amplitude 1 [volt]. A frequźncia é baixa o
suficiente de modo a simular a repetiēćo de degraus na entrada.
\end{par}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
\section{Montagem experimental}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%

\begin{par}
Para o procedimento experimental, foi utilizada a bancada Elvis para aquisiēćo dos sinais do
circuito, seguindo as instruēões de montagem apresentadas no roteiro. Logo após, foi utilizado 
o software de aquisiēćo LabView, para aplicar uma entrada de onda quadrada com frequźncia 
$f_e = 1 [Hz]$ e amplitude $V = 1 [V]$, e ler a resposta em cada um dos estįgios.
\end{par}

\subsection{Estįgio 1}
\begin{par}
Para o primeiro estįgio, era esperado nćo haver dināmica na resposta. Ao fazer a leitura, 
foi encontrado um grande ruķdo, que pode ser identificado na Figura \ref{fig:est1}.
\end{par}
\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est1_leitura.png}
\caption{Leitura feita para o primeiro estįgio.}
\label{fig:est1}
\end{figure}
\begin{par}
Sabe-se de imediato que os pontos próximos de zero podem ser considerados zero. 
Basta entćo descobrir qual o valor médio dos valores abaixo. Uma maneira interessante 
é o uso de mķnimos quadrados para os valores abaixo de zero. O código abaixo resolve esse 
problema de forma simples, e exibe seus resultados na Figura \ref{fig:est1minquad}. 
O ganho encontrado é portanto de $-0.1011 \pm 0.0198$, colocando o ganho esperado dentro 
de sua margem de erro.
\end{par}

\begin{verbatim}
% Pega a parte debaixo do degrau
indEst1 = find(est1<-0.04);
est1l_min = est1(indEst1);
trandom = 1:length(est1l_min);
figure, scatter(trandom,est1l_min,'x')
hold on

% fit simples do vetor.
[fit1,weight]=polyfit(trandom',est1l_min,0);
[fit1p,delta] = polyval (fit1,trandom,weight);
plot(trandom,fit1p,'r')
\end{verbatim}

\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est1_minquad.png}
\caption{Leitura feita para o primeiro estįgio.}
\label{fig:est1minquad}
\end{figure}

\subsection{Estįgio 2}
\label{subsec:est2}

\begin{par}
Pode-se perceber pela Figura \ref{fig:est2} que hį um grande ruķdo que nos impede de inferir os valores de ganho apropriadamente. 
Uma forma interessante de filtrar o ruķdo pode ser alcanēada observando dois fatores: a leitura foi 
feita com uma frequźncia de 1kHz, e a frequźncia da rede de distribuiēćo (causa mais comum de ruķdos 
elétricos) é de 60Hz.
\end{par}

\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est2_leitura.png}
\caption{Leitura feita para o segundo estįgio.}
\label{fig:est2}
\end{figure}

\begin{par}
Tendo essas informaēões em vista, pode ser aplicado um filtro discreto de média rolante sobre o sinal 
de forma a eliminar o ruķdo. A idéia é fazer a média de um ciclo inteiro de 60Hz, eliminando assim o ruķdo. 
Assim, basta escolher a quantidade da pontos necessįrios para tal:
\end{par}

\begin{equation} \label{eq:pontosRuido}
N°\, de\, pontos = \frac{Freq.\, Leitura}{Freq.\, Ruido} = \frac{1000}{60} = 16.667\, pontos
\end{equation}

\begin{par}
Escolheu-se entćo $N = 17$ pontos para filtrar o sinal e adquirir uma boa aproximaēćo. A partir dessa leitura, 
foi possķvel encontrar a aproximaēćo exibida na Figura{fig:est2filtro}. 
\end{par}

\begin{verbatim}
figure, plot(t2,est2);
hold on
plot(t2,smooth(est2,17),'r');
\end{verbatim}

\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est2_smooth.png}
\caption{Filtro feito para o segundo estįgio.}
\label{fig:est2filtro}
\end{figure}

\begin{par}
A partir dos valores mostrados na figura \ref{fig:est2filtro}, pode-se entćo estimar o valor de ganho e 
também obter o valor de $\tau_2$ a ser encontrado, de acordo com a equaēćo:
\end{par}

\begin{equation*} \label{eq:tau2}
\tau_2 = \frac{t}{4},\, em\, y(t)= 98\%\, _{regime}
\end{equation*}
\begin{equation} \label{eq:tau2}
\tau_2 = 0.165
\end{equation}

\subsection{Estįgio 3}

\begin{par}
Os dados experimentais encontrados podem ser analisados na figura \ref{fig:est3_leitura}. 
Utilizando o mesmo procedimento de filtragem apresentado no sub-tópico \ref{subsec:est2}, pode-se encontrar 
a curva apresentada na Figura \ref{fig:est3_filtrado}.
\end{par}

\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est3_leitura.png}
\caption{Filtro feito para o segundo estįgio.}
\label{fig:est3_leitura}
\end{figure}

\begin{figure}[H]
\centering
\includegraphics [width=4in]{Relatorio/est3_smooth.png}
\caption{Filtro feito para o segundo estįgio.}
\label{fig:est3_filtrado}
\end{figure}

\begin{par}
O valor de $\tau_3$ pode ser encontrado utilizando a seguinte equaēćo:
\end{par}

\begin{equation} \label{eq:tau2}
x^2 - \frac{1-b-\exp{^-2}}{1-a-\exp{^-1}} x + \frac{(1-b)exp{^-1} - (1-a)\exp{^-2}}{1-a-\exp{^-1}}=0
\end{equation}
\begin{par}
Onde:
\end{par}

\begin{equation*}  
a = y(\tau_2)/k
\end{equation*}  
\begin{equation*}  
b = y(2\tau_2)/k
\end{equation*}  

\begin{par}
Resolvendo para ambos os casos pelo código abaixo, e utilizando para isso 
os pontos mostrados na Figura \ref{fig:est3_filtrado}:
\end{par}

\begin{verbatim}
k3 = -0.9854;
a = -0.1105 / k3;
b = -0.4372 / k3;
poly = [1 -(1-b-exp(-2))/(1-a-exp(-1)) ((1-b)*exp(-1)-(1-a)*exp(-2))/(1-a-exp(-1)) ];
rpoly = roots(poly);
tau3 = rpoly(1)
\end{verbatim}

\begin{equation*}
\tau_3 = \frac{-\tau_2}{ln(x)}
\end{equation*}

\begin{equation}
\tau_3 = 0.4417
\end{equation}
